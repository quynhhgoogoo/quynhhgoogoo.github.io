Course Scheduling

Using JavaScript, MySQL, AJAX, HTML5, CSS3 and Python Django Framework.

The solution will also enable calculation, how many hours totally the teachers are teaching and how many lessons there is for an individual week .The databases are stored in MyPHP Admin named "e1601128_course_new"

You should install pip and python environment. Before using, you can kindly clone it from my github, open cmd and run "python manage.py runserver"

1. The main page of this software can be accessed via this page : https://localhost:8000/coursescheduling.html

2. The sql querries can be found in : sqlquerries.txt

3. The set up files are : run.bat and setup.bat

4. Now, you are able to search, modify and do several impacts on my database
